# loic_npm
Adds the ability to manipulate a Loic object.

# Instalation

run

    npm install git://github.com/nkcr/loic_npm.git

# Usage

Via the console :

    node
    > var Loic = require('loic_npm');
    > var loic = new Loic();
    > loic.shut_up();
    Shut up Loic!
    > loic.is_shut();
    Loic is currently shut

Pretty amazing eh?
